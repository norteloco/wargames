# Что такое?

На сайте [owerthewire.org](https://overthewire.org/) представлено некоторое количество игр (wargames) для исследования собственных навыков в операционных системах и информационной безопасности.

## Какие игры есть?

Согласно описанию с сайта overthewire.org:

1. Bandit — основы Unix/Linux
2. Natas — веб-безопасность
3. Krypton — криптография
4. Leviathan — реверс-инжиниринг
5. Narnia — начало бинарной эксплуатации (и реверс-инжиниринга)
6. Behemoth — бинарная эксплуатация (и реверс-инжиниринг)
7. Utumno — бинарная эксплуатация (и реверс-инжиниринг)
8. Maze — бинарная эксплуатация (и реверс-инжиниринг)  
и т.д.  

## Как проходить? 

У каждой игры есть собственное описание. Каждый уровень усложняется постепенно, хотя могут быть встречены уровни, которые могут показаться вам значительно сложнее, или наоборот проще, чем те, которые вы уже прошли.
